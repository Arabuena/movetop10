Este placeholder é carregado quando a URL remota não responde.

Para usar o app offline, gere o build do frontend e copie os arquivos para android/app/src/main/assets/.

  1. Na pasta frontend: npm run build
  2. Copie o conteúdo de frontend/build para android/app/src/main/assets
  3. Reinstale o APK

Para configurar a URL remota via JavaScript use: window.Android.setBaseUrl('http://SEU_IP:3030')
