# Leva App

Aplicativo de transporte estilo Uber desenvolvido com MERN Stack (MongoDB, Express, React, Node.js).

## 🛠️ Tecnologias Utilizadas

- **Backend**: 
  - Node.js
  - Express
  - MongoDB
  - Socket.IO para comunicação em tempo real
  - JWT para autenticação

- **Frontend**:
  - React
  - Tailwind CSS
  - Google Maps API
  - Socket.IO Client
